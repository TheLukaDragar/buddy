# WORKOUT PROGRAMMING GUIDE - BASE WEEK of 8 week program GENERATOR (v6.0)

## 1. GENERAL PRINCIPLES

‚úÖ Generate one foundational training week that can be repeated or modified
‚úÖ Use full gym equipment when available
‚úÖ Respect client goals and muscle-group preferences
‚ùå Never mix upper and lower body in a single exercise
‚ùå No supersets or circuit training
‚ùå Warm-up is never part of the workout plan
‚ùå Cardio equipment is never included

## 2. EXECUTION STANDARDS

**Rest & Timing:**
- Rest between sets = 90 seconds
- Set duration = ~60 seconds of work
- Each exercise = 4 sets = ~10 minutes total

**Push-Up Rules:**
‚úÖ Only flat-hand or feet-elevated variations
‚ùå Never use "incline push-ups"

**Substitutions:**
- Bench Press ‚Üí Push-Ups (Regular or Feet-Elevated)
- Incline Bench ‚Üí Feet-Elevated Push-Ups

**Range of Motion:**
‚úÖ Increase ROM when appropriate
‚ùå Never increase ROM for shoulder-based exercises

**Isolation:**
‚ùå No isolation exercises without external load

**Unilateral Movements:**
- Side-lying exercises (Left + Right) = ONE exercise entry
‚ùå Never separate sides into two entries

## 3. BEGINNER ADAPTATION

Use if client is new to training or returning after break:
- Reps: 10‚Äì15
- Intensity: Low to moderate  
- Same exercises as standard plan
- Avoid explosive or high-complexity movements

## 8. EXERCISE SPACING REQUIREMENTS

**Critical Spacing Rules:**
- Deadlift ‚Üî Squat = 1+ full rest day between
- Deadlift ‚Üí Deadlift = 2+ days apart  
- Squat ‚Üí Squat = 2+ days apart

**Fallback Rule:**
‚úÖ Use machine-based alternatives if spacing impossible (Leg Press, Leg Curl, Leg Extension)
‚ùå Never ignore spacing rules for program convenience

## 9. FULL-BODY DAY REQUIREMENTS

Each full-body session MUST include:
- ‚úÖ One Push exercise
- ‚úÖ One Pull exercise  
- ‚úÖ One Leg exercise
- ‚úÖ One Core exercise

**50% Focus Rule:** May apply to maximum ONE full-body day per week

## 10. SPECIAL EXERCISE RULES

**Deadlifts:** All barbell deadlifts = Leg/Glute category, NOT Back

**Back Exercises:** Pulldowns, pull-ups, rows, cable pulls
‚ùå Back Extensions = glute/hamstring, not back category

**Isometric Core:** Only these exercises use time instead of reps

| Exercise | Time | Notes |
|----------|------|-------|
| Plank | 30 sec | Keep core tight |
| Side Plank | 30 sec | Keep core tight |
| Bird Dog Hold | 30 sec | Keep core tight |
| Plank Shoulder Taps | 30 sec | Keep core tight |

**Leg Curls:** Treat as isotonic - fill reps, leave time blank

## 11. STREAK DAYS (HOME ALTERNATIVES)

Maximum 3 streak days can be designated per base week.

**Requirements:**
‚úÖ Must match gym workout: same number of exercises, biomechanics, target muscles
‚úÖ Use only home-available equipment from client's list
‚úÖ If no home equipment listed, bodyweight only
‚ùå Never assume dumbbells, bands, benches, or chairs
‚ùå Never duplicate streak exercises within same session

## 13. VALIDATION CHECKLIST

Before generating plan, verify:
- ‚úÖ Deadlift/squat spacing requirements met
- ‚úÖ Full-body days include Push/Pull/Leg/Core
- ‚úÖ No exercise appears more than once in the base week
- ‚úÖ Streak exercises are unique per session
- ‚úÖ Streak equipment matches client's home setup only
- ‚úÖ Training days follow chronological order
- ‚úÖ Frequency matches appropriate split protocol

## 14. ADVANCED USER ADAPTATIONS

For experienced clients:
‚úÖ Include unilateral and complex compound movements
‚úÖ Add tempo/intensity variations in notes
‚úÖ Use lower rep ranges for compound movements
‚úÖ Emphasize progressive overload principles

## 15. LOW-SORENESS CLIENTS

Emphasize:
‚úÖ Machine-based exercises
‚úÖ Core strengthening
‚úÖ Posterior chain work  
‚úÖ Controlled movements
‚ùå Avoid excessive eccentric loading early in week
